# Senny's Portfolio

This is my portfolio project, a small corner of the internet for showing my experience, selected work, and whatever I am currently shaping into something presentable.

It is mainly here for recruiters who want a quick sense of what I do and how I think. Clients are welcome too. Future me is also absolutely invited, probably arriving with too many tabs open and coffee nearby.

## What's Inside

- A portfolio-style frontend built with React and Vite
- Project and experience sections for showing the work
- A Node-friendly setup for local development
- A personal playground vibe, because portfolios are never really done

## Tech

- React
- Vite
- TypeScript
- Tailwind CSS
- Node.js

## Running It Locally

First install the dependencies:

```bash
npm install
```

Then start the dev server:

```bash
npm run dev
```

Vite will print the local URL in the terminal, usually something like:

```bash
http://localhost:5173/
```

## Other Useful Commands

Build the project:

```bash
npm run build
```

Run linting:

```bash
npm run lint
```

Preview the production build:

```bash
npm run preview
```
